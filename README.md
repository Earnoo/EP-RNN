# Overview

Despite their strong sequential modeling capabilities, recurrent neural networks (RNNs) remain vulnerable to noise, outlier-contaminated, and missing values in real-world time series, which undermines robustness and generalization. Conventional forecasting models struggle when inputs are corrupted, missing, or contaminated by outliers. Moreover, existing approaches that inject innovation signals either tie them to the output head, complicating training, or fail to provide gate-level corrective signals for broadly applicable forecasting. To address these limitations, we propose Error-Projected RNNs (EP-RNNs), a lightweight, architecture-agnostic modification that augments gated cells with a hidden-space innovation signal. Each gate receives a learnable corrective signal derived from the difference between the current input aligned proposal and the carried hidden state, decoupled from the output head and fully compatible with standard backpropagation through time. This mechanism enhances robustness, adaptability, and predictive accuracy under noise, missingness, outliers, and distribution shifts. We construct synthetic stress tests as controlled perturbation suites that add noise, induce missing values, and insert outliers, enabling us to evaluate how projected error aligns with realized future error and how it influences gate weights. We then validate EP-RNNs on these stress tests and on three real-world domains, including traffic flow, influenza-like illness, and stock market forecasting. Experimental results, evaluated using MSE, RMSE, and MAPE, demonstrate substantial and consistent improvements over standard LSTM, GRU, and BiLSTM models. Specifically, EP-LSTM achieves up to 83% improvement, EP-GRU up to 41%, and EP-BiLSTM up to 76% improvement on real-world datasets, highlighting the practical effectiveness of gate-level error modulation for reliable time series forecasting.
